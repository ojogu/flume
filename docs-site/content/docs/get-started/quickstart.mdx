# Quickstart <span className="text-display text-brand font-normal ml-1">Journey</span>

Complete a media processing workflow in under five minutes using our canonical HTTP API or official SDKs.

<Steps>
<Step>
### Install the SDK
Install the official Flume SDK or use direct API calls. Sandbox keys are ready for use.

<Tabs items={['HTTP (cURL)', 'Node.js', 'Python', 'Go']}>
<Tab value="HTTP (cURL)">
```bash
# No installation required for HTTP API.
curl --version
```
</Tab>
<Tab value="Node.js">
```bash
npm install @flume/node
```
</Tab>
<Tab value="Python">
```bash
pip install flume-python
```
</Tab>
<Tab value="Go">
```bash
go get github.com/flume-media/flume-go
```
</Tab>
</Tabs>
</Step>

<Step>
### Initialize & Authenticate
Authenticate your requests using the `X-API-Key` header. Use your sandbox key (`fl_test_...`) to initialize the client without incurring any costs.

<Tabs items={['HTTP (cURL)', 'Node.js', 'Python', 'Go']}>
<Tab value="HTTP (cURL)">
```bash
curl -H "X-API-Key: fl_test_..." \
  https://api.flume.ojogulabs.xyz/v1/auth/verify
```
</Tab>
<Tab value="Node.js">
```javascript
import { Flume } from '@flume/node';

const flume = new Flume('fl_test_...');
```
</Tab>
<Tab value="Python">
```python
from flume import Flume

flume = Flume('fl_test_...')
```
</Tab>
<Tab value="Go">
```go
import "github.com/flume-media/flume-go"

client := flume.NewClient("fl_test_...")
```
</Tab>
</Tabs>
</Step>

<Step>
### Send Your First Job
Define a pipeline of operations to transform your source media. We'll trim the first 10 seconds and extract the audio.

<Tabs items={['HTTP (cURL)', 'Node.js', 'Python', 'Go']}>
<Tab value="HTTP (cURL)">
```bash
curl -X POST https://api.flume.ojogulabs.xyz/v1/job \
  -H "X-API-Key: fl_test_..." \
  -H "Content-Type: application/json" \
  -d '{
    "source": "art_UPLOAD_ID",
    "pipeline": [
      { "operation": "trim", "params": { "start": 0, "end": 10 } },
      { "operation": "extract_audio" }
    ]
  }'
```
</Tab>
<Tab value="Node.js">
```javascript
const job = await flume.jobs.create({
  source: 'art_UPLOAD_ID',
  pipeline: [
    { operation: 'trim', params: { start: 0, end: 10 } },
    { operation: 'extract_audio' }
  ]
});
```
</Tab>
<Tab value="Python">
```python
job = flume.jobs.create(
    source="art_UPLOAD_ID",
    pipeline=[
        {"operation": "trim", "params": {"start": 0, "end": 10}},
        {"operation": "extract_audio"}
    ]
)
```
</Tab>
<Tab value="Go">
```go
job, _ := client.Jobs.Create(context.Background(), flume.JobCreateRequest{
    Source: "art_UPLOAD_ID",
    Pipeline: []flume.Operation{
        {Name: "trim", Params: map[string]interface{}{"start": 0, "end": 10}},
        {Name: "extract_audio"},
    },
})
```
</Tab>
</Tabs>
</Step>

<Step>
### Monitor Progress
Processing is asynchronous. Poll the job status to know when your results are ready.

<Tabs items={['HTTP (cURL)', 'Node.js', 'Python', 'Go']}>
<Tab value="HTTP (cURL)">
```bash
curl -H "X-API-Key: fl_test_..." \
  https://api.flume.ojogulabs.xyz/v1/job/JOB_ID
```
</Tab>
<Tab value="Node.js">
```javascript
const status = await flume.jobs.get(job.id);
console.log(status.status); // 'completed'
```
</Tab>
<Tab value="Python">
```python
status = flume.jobs.get(job.id)
print(status.status) # 'completed'
```
</Tab>
<Tab value="Go">
```go
status, _ := client.Jobs.Get(context.Background(), job.ID)
fmt.Println(status.Status)
```
</Tab>
</Tabs>
</Step>
