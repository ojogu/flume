# Rate Limiting

The Flume API enforces rate limits to ensure stability and fair usage across all consumers.

## Default Limits
- **Free Tier**: 100 requests per minute.
- **Pro Tier**: 1000 requests per minute.
- **Enterprise**: Custom limits based on your agreement.

## Handling Limits
When a rate limit is exceeded, the API returns a `429 Too Many Requests` response.
Check the `X-RateLimit-Reset` header to see when you can resume making requests.
